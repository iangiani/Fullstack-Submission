# Part 3 Phonebook

# 3.10
- Backend (Render): [https://fullstack-submission-qoz8.onrender.com/api/persons](https://fullstack-submission-qoz8.onrender.com/api/persons)

## Description

- `GET /api/persons` — return all persons  
- `GET /api/persons/:id` — return one person  
- `POST /api/persons` — add a new person  
- `DELETE /api/persons/:id` — remove a person  

## Run locally

git clone https://github.com/iangiani/Fullstack-Submission.git
cd Fullstack-Submission/part3/phonebook
npm install
